解决Windows家庭版无法远程的问题

要解决这个问题需要一个工具rdpwrap：下载地址 | Github

下载回来之后解压，可以看到：

以管理员的方式运行install.bat

运行里面的RDPConf.exe，看是否有红色的标志。如果有的话，就需要查看自己系统的C:\Windows\System32\termsrv.dll的版本号：

到这里搜索该版本号的配置文件，仔细找，应该能找到像下面的提供的ini文件

找到后，请先win+r运行：services.msc，停止Remote Desktop Services服务，

将下载的ini文件覆盖到C:\Program Files\RDP Wrapper里面

重新开启Remote Desktop Services服务，

运行检查程序看看是不是全绿，如果是全绿色的话，就证明现在可以使用远程了，win+r运行，输入mstsc打开远程桌面窗口。
